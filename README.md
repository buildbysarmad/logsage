
# LogSage

Paste your logs. Know what broke and why. In seconds.

🌐 [logsage.dev](https://logsage.dev)

## What it does

LogSage automatically detects your log format, groups similar errors
by pattern, and surfaces root causes — so you fix things faster
without manually scanning thousands of lines.

Supported formats: Serilog, NLog, Log4Net, and plain text logs.
